# MyNetwork

A Last Minute Networking App

## Overview

**MyNetwork** is a TypeScript-based networking application designed to facilitate quick and efficient connection-building among professionals. Whether you're attending an event, joining a new community, or looking to expand your professional circle, MyNetwork provides an intuitive platform for making meaningful connections at the last minute.

## Features

- **Quick Connection Creation**: Rapidly build and manage your professional network
- **Last-Minute Availability**: Easily indicate your availability for networking events
- **TypeScript Foundation**: Built with modern TypeScript for type safety and reliability
- **Scalable Architecture**: Designed to handle growing user bases and expanding features

## Tech Stack

- **Language**: TypeScript (99.8%)
- **Build System**: Modern JavaScript/TypeScript tooling
- **Platform**: Built on AI Studio repository template

## Getting Started

### Prerequisites

- Node.js 16 or higher
- npm or yarn package manager
- Gemini API Key (for AI features)

### Installation

```bash
# Clone the repository
git clone https://github.com/anudeep0011/MyNetwork.git

# Navigate to the project directory
cd MyNetwork

# Install dependencies
npm install
```

### Configuration

1. Create or update your `.env.local` file
2. Add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

### Running the Application

```bash
# Development mode
npm run dev

# Production build
npm run build

# Start production server
npm start
```

The app will be available at `http://localhost:3000` (or the configured port).
